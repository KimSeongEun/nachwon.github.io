### Test Driven Development with Django 준비 사항

- Python 버전 3.4 이상

- 기본적인 `HTML` 지식

- 약간의 `Javascript` 지식

- 기본적인 `Vi` 텍스트 에디터 사용법


- 필요한 소프트웨어
    - 웹 브라우저: 구글 크롬 또는 파이어폭스
    - Git
    - pip Python 패키지 매니저


- 필요한 Python 패키지
    - Django
    ```
    pip install django
    ```

    - Selenium
    ```
    pip install --upgrade selenium
    ```
